# 1.0.4 Codex CLI 선택 설치

## 문제

Codex 데스크톱 앱과 독립 실행 Codex CLI의 차이를 모르는 사용자는 모니터 설치를 마친 뒤에도 Setup에서 `CLI 없음`을 보게 된다. 설치 안내를 다시 열어 별도 설치를 마쳐야 하므로 첫 실행이 쉽게 막혔다.

## 결정

- Tauri NSIS의 `NSIS_HOOK_POSTINSTALL`에서 독립 실행 CLI 설치 여부를 확인한다.
- CLI가 없고 대화형 설치일 때만 OpenAI 공식 Windows 설치 프로그램을 실행할지 묻는다.
- 기본 선택은 **아니요**로 두고 사용자가 동의한 경우에만 공식 `chatgpt.com` 스크립트를 실행한다.
- Codex CLI 바이너리를 모니터 설치 파일에 포함하지 않는다.
- 거절 또는 실패는 모니터 설치를 실패시키지 않는다.
- 무인 설치(`/S`)에서는 질문과 네트워크를 건너뛴다.
- 계정 로그인은 설치 프로그램이 자동 실행하지 않고 앱의 Setup에서 사용자가 직접 시작한다.

## 검증 기준

1. 기존 OpenAI 독립 실행 설치본 또는 npm 전역 설치본이 있으면 질문하지 않는다.
2. 동의 대화상자의 기본 버튼은 **아니요**다.
3. 공식 Windows 명령인 `powershell -ExecutionPolicy ByPass -c "irm https://chatgpt.com/codex/install.ps1 | iex"`과 같은 스크립트 URL을 사용한다.
4. `/S` 설치는 사용자 입력과 네트워크 없이 끝난다.
5. CLI 설치 종료 코드와 관계없이 모니터 설치는 완료된다.
